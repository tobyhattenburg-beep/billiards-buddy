# Content rating (IARC) — questionnaire notes

Play Console → App content → Content rating. Category: **Reference, News, or Educational** / **Utility** (or Social if prompted — the app is a sports companion with a chat feature).

Answer honestly; this app is low-risk. Key points:

- Violence / sexual content / nudity / profanity in the app's own content: **No** (the app ships none; a client-side profanity filter blocks it in chat).
- Controlled substances / gambling: **No** (no real-money gambling; tournament "prizes" are user-defined text, no in-app wagering).
- **Users can interact / communicate with each other: YES.** The Community chat lets users exchange messages. Declare this — it typically adds a "Users interact" / "Shares user-provided content" descriptor.
- User-generated content sharing: **Yes** (chat messages, highlights). Note the moderation in place: profanity filter, per-message **report**, and per-user **block** (in-app), plus contact email for takedown requests.
- Shares user's location with other users: **No** (location is used only to find venues, not broadcast to other users).
- In-app purchases: **No** (affiliate links to external retailers; no IAP).

Expected rating: everyone / teen-ish due to open chat. Whatever IARC returns, ensure the chat moderation tools above are mentioned so the UGC policy is satisfied.
